## react+react-router+redux 移动端SPA webpack版本

### 安装

npm i --registry=https://registry.npm.taobao.org

### 主要包括能力：

- 基于webpack的编译
- 基于express的本地服务，并增加mock能力
- 支持热加载
- flexible移动端布局

### 接入`browser-sync` 

极大方便移动端调试，同时支持`mock`能力！
